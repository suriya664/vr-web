# Obsidian Reality - Enterprise VR Solutions Website

A luxury, enterprise-grade website built with HTML, CSS, JavaScript, Bootstrap, AJAX, and animations.

## Theme: Obsidian Reality

**Vibe:** Luxury • Enterprise • Powerful

**Color Palette:**
- Deep black (#000000, #0a0a0a)
- Graphite gray (#2a2a2a, #3a3a3a, #4a4a4a)
- Platinum (#E5E4E2)
- Subtle gold (#D4AF37, #C9A961)

## Features

- ✅ Responsive design (mobile, tablet, desktop)
- ✅ Semantic HTML5 structure
- ✅ Modern CSS with animations
- ✅ Form validation with AJAX
- ✅ Smooth scrolling
- ✅ Active page highlighting
- ✅ Mobile-friendly navigation
- ✅ Social media integration
- ✅ SEO optimized

## Getting Started

1. Open `index.html` in a web browser
2. All pages are linked and functional
3. Forms include validation and AJAX simulation

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- Add your favicon to `images/favicon.ico`
- Update contact information in footer and contact page
- Configure form submission endpoints in `js/main.js`
- Add your images to the `images/` folder
